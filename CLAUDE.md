# CLAUDE.md

## 仓库说明

本仓库用于跟踪 Arxiv 上的论文进展。各领域论文按子目录分类管理。

## 工作流程

1. 使用 `/arxiv-search` skill 检索指定领域的论文，生成论文简述
2. 将论文简述更新到对应领域的 markdown 文档中（如 `3dgs/3dgs_tracking.md`）
3. 提交更改并推送到远程仓库

## 文档格式规范

每个领域的 markdown 文档包含两部分：

### 目录（表格）

以表格形式列出所有论文，格式如下：

```markdown
| 日期 | 论文 | arXiv | 类别 | 简述 |
|------|------|-------|------|------|
| 2026-05-18 | [PanoWorld](#锚点) | [2605.17916](https://arxiv.org/abs/2605.17916) | cs.CV | 一句话简述 |
```

锚点格式：小写英文短横线分隔，如 `1-panoworld-a-generative-spatial-world-model...`

### 论文简述

按日期分组，每篇论文格式如下：

```markdown
## YYYY-MM-DD

### 1. 论文标题

- **arXiv**: [编号](https://arxiv.org/abs/编号)
- **作者**: 作者列表
- **类别**: 分类（如 cs.CV）
- **备注**: （可选，如录用信息）

**摘要**: 中文摘要，简述研究背景、方法、结果。
```

多篇论文之间用 `---` 分隔。

详细格式示例请参考 `3dgs/3dgs_tracking.md`。
