# 分类文档规范

月度论文收集完成后，需要编写分类文档，按研究方向对论文进行归类。

## 工作流程

1. 复制 `category_template.md` 为 `<领域>_category_YYYY-MM.md`，放置在同领域子目录中
2. 将标题和说明中的 `YYYY-MM` 替换为对应月份
3. 从月度文件中读取论文，按研究方向填入对应类别的表格
4. 更新底部的统计概览

## 文档格式

- 表格列：`| 日期 | 论文 | arXiv | 关键词 |`
- 每个分类包含：表格 + 概要段落
- 表格行按日期从近到远（降序）排列

## 分类规则

- 论文按最匹配的研究方向归类，一篇论文只归一个主类别
- 若论文跨多个领域，归入最核心的研究方向
- 若某分类本月无论文，删除该分类（不保留空分类）
- 分类编号使用中文数字（一、二、三...），删除后重新编号

## 锚点要求

- 标题后添加显式 HTML 锚点 `{#id}`，格式如 `## 一、核心算法与优化 {#cat-core-opt}`
- 目录链接与标题锚点保持一致
- 论文链接指向月度文件中的对应锚点

## 亮点论文标记

与月度文件相同，顶会/顶刊录用论文格式为 `★ **[论文名](#锚点)**`，关键词列末尾标注会议信息（如 `CVPR'26`、`SIGGRAPH'26`）。

## 统计概览

文档底部统计表格只列出实际有论文的类别，论文数按模板中实际数量填写。
