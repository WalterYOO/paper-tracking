# 分类文档规范

月度论文收集完成后，需要编写分类文档，按研究方向对论文进行归类。

## 文档格式

- 表格列：`| 日期 | 论文 | arXiv | 关键词 |`
- 每个分类包含：表格 + 概要段落
- 表格行按日期从近到远（降序）排列

## 分类规则

- 一篇论文可归入多个类别（不局限于一个），按论文实际涉及的研究方向分别归入
- 若某分类本月无论文，删除该分类（不保留空分类）
- 分类编号使用中文数字（一、二、三...），删除后重新编号

## 锚点要求

- 标题后添加显式 HTML 锚点 `{#id}`，格式如 `## 一、核心算法与优化 {#cat-core-opt}`
- 目录链接与标题锚点保持一致
- 论文链接指向月度文件中的对应锚点

## 亮点论文标记

与月度文件相同，顶会/顶刊录用论文格式为 `★ **[论文名](#锚点)**`，关键词列末尾标注会议信息（如 `CVPR'26`、`SIGGRAPH'26`）。

## 统计概览

文档底部统计表格只列出实际有论文的类别，论文数按模板中实际数量填写。
